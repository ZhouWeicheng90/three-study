#### 颜色光照的原理

```js
// 漫反射数学模型RGB分量表示：(R2,G2,B2) = (R1,G1,B1) x (R0,G0,B0) x cosθ
// 相乘的两种颜色分别为 材质颜色和光线颜色，Θ为入射角
R2 = R1 * R0 * cosθ
G2 = G1 * G0 * cosθ
B2 = B1 * B0 * cosθ
// 漫反射下，黑色的 永远是黑色的、蓝光照红mesh也是黑的……
```
